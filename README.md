Nombre del proyecto asignado: Gestor de reportes con distintos formatos de salida

Descripción: Este programa en C# simula la decoración de un reporte credo por consola utilizando el patrón de diseño decorador (Decorator), además, de "poder exportar" el reporte en formato de documento (PDF, Excel, Word, PowerPoint) o imagen (JPG, PNG) utilizando el patrón compuesto (Composite), incluyendo el patron método fábrica y el patró de arquitectura en capas.

Descargar el zip del repositorio, una vez descargado descomprimir la carpeta, despues abrir la carpeta "U4U5Examen" que está dentro de "SRC" y dar doble clic en la solución (U4U5Examen.sln). Una vez inicializado el programa seleccionar la opción "Iniciar" para ejecutar el código.
